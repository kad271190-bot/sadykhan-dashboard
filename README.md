# SADYKHAN — AI Dashboard управления репутацией 2GIS

Production-ready Next.js 14 дашборд для мониторинга и управления репутацией аптечной сети SADYKHAN.

## Стек

- **Next.js 14** (App Router)
- **TypeScript** — строгая типизация
- **Tailwind CSS** — стилизация
- **Recharts** — графики и аналитика
- **Lucide React** — иконки
- **date-fns** — форматирование дат

## Быстрый старт

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Деплой на Vercel (получить ссылку за 2 минуты)

```bash
# 1. Установить Vercel CLI
npm i -g vercel

# 2. Задеплоить
vercel

# 3. В браузере откроется ваш URL вида:
# https://sadykhan-dashboard-xxx.vercel.app
```

## Деплой на Netlify

```bash
npm run build
# Загрузить папку .next на Netlify через drag-and-drop
# или подключить GitHub репозиторий
```

## Функциональность MVP

### ✅ Реализовано
- 8 ключевых метрик репутации в реальном времени
- Критические алерты с раскрывающимся списком
- Таблица 15+ филиалов с сортировкой по рейтингу, отзывам, Health Score
- Фильтр по городу (Алматы / Астана / Тараз)
- Лента отзывов из 2GIS с классификацией по категориям и критичности
- AI-генератор ответов (3 варианта: эмпатичный / официальный / краткий)
- Графики динамики рейтинга за 12 месяцев (Recharts)
- График негативных отзывов по месяцам
- Горизонтальный бар-чарт топ категорий жалоб
- Тепловая карта жалоб по дням недели
- Сравнение с 5 конкурентами
- AI-инсайты на основе реальных данных 2GIS
- Коллапсируемая боковая навигация
- Поиск по отзывам и филиалам
